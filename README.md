# Food Delivery Data Analysis

## Overview
This project demonstrates how to combine CSV, JSON, and SQL datasets using Python in Google Colab.

## Files Used
- orders.csv (Transactional data)
- users.json (User master data)
- restaurants.sql (Restaurant master data)

## Output
- final_food_delivery_dataset.csv

## Tools & Technologies
- Python
- Pandas
- SQLite
- Google Colab

## Key Analysis
- City-wise revenue
- Cuisine-wise performance
- Gold vs Regular membership analysis
- Seasonal revenue trends
